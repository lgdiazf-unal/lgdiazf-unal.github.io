## Instalacion

### 1. Actulizar los paquetes

```bash
sudo yum update
```

### 2. Instalar postgresql

```bash
yum install @postgresql
```

### 3. Iniciar y habilitar el servicio

```bash
systemctl start postgresql
systemctl enable postgresql
```

### 4. Configurar accesos de red

Por defecto postgres solo es accesible desde la red interna de la maquina, para que clientes externos puedan acceder se debe modificar el archivo ***/var/lib/pgsql/data/postgresql.conf***, descomentar la linea `listen_addresses` y permitir todo el trafico:

```bash
listen_addresses = *
```
